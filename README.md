# palv2

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```

Or install bun project in global scope then execute:

```bash
bun link
bun link pal
# example command
pal exec --path hello
```

This project was created using `bun init` in bun v1.2.12. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
